Referensi Belajar HTML dfn tag. Tutorial dan panduan mengenai element <dfn>...</dfn> yang digunakan untuk mendefinisikan sebuah istilah. Mencakup penjelasan yang disertai contoh kode penggunaan sebagai rujukan untuk materi belajar HTML elemen <dfn>.

Deskripsi
Penjelasan HTML dfn

HTML <dfn> tag digunakan untuk mendefinisikan perihal sebuah istilah (term).

Sebuah istilah yang akan dijelaskan pengertiannya dapat ditulis didalam element <dfn>, yang mana <dfn> itu sendiri harus terletak didalam element lain yang manaunginya, diantaranya element <p>, <section> atau <dt>.

Penjelasan dari istilah ditulis secara langsung setelah <dftn> element dan terletak didalam element yang menaunginya tersebut.

Global Attributes
Atribut Secara Global (Keseluruhan)

<dfn> tag mencakup global attributes, yang artinya tag tersebut dapat disisipkan semua attributes yang termasuk dalam global attributes yang secara umum berlaku untuk semua HTML tags.

Silahkan, lihat referensi mengenai HTML Global Attribute

Event Attributes
Atribut event (Peristiwa)

<dfn> tag mencakup event attributes, yang artinya attribute tersebut dijalankan ketika ada interaksi dari user atau dalam suatu peristiwa (kejadian). Contoh: menjalankan script (JavaScript) ketika halaman web pada jendela browser hendak ditutup, dan lain sebagainya.

Silahkan, lihat referensi mengenai HTML Events Attribute

Example
Contoh HTML <dfn> element

Contoh 1

Contoh <dfn> element yang menjelaskan defini dari "HTML"

HTML 
<p>
  <dfn>HTML</dfn> adalah bahasa markup yang digunakan untuk memformat struktur halaman web.
</p>
Contoh 2

Contoh <dfn> element dengan attribute title

HTML 
<p>
  <dfn title="HyperText Markup Language">HTML</dfn> adalah bahasa markup yang digunakan untuk memformat struktur halaman web.
</p>

Contoh <dfn> element dengan attribute title langsung didalam element <dfn> itu sendiri.

Contoh 3

Contoh <dfn> dengan <abbr> element didalamnya.

HTML 
<p>
  <dfn>
    <abbr title="HyperText Markup Language">
      HTML
    </abbr>
  </dfn>
  adalah bahasa markup yang digunakan untuk memformat struktur halaman web.
</p>

Contoh <dfn> dengan <abbr> element didalamnya dan menggunakan attribute title untuk istilah (term) yang akan didefinisikan.

Contoh 4

Contoh lengkap <dfn> element dengan attribute id

HTML 
<p>
  <dfn id="definisi-html">
    <abbr title="HyperText Markup Language">
      HTML
    </abbr>
    </dfn>
  adalah bahasa markup yang digunakan untuk memformat struktur halaman web.
</p>

Contoh <dfn> element menggunakan id, sehingga bisa digunakan untuk mereferensikan definisi yang merujuk pada id tersebut menggunakanan <a> element.
