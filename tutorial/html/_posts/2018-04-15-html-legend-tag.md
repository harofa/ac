<p class="text-muted">
    Referensi Belajar <strong>HTML legend tag</strong>. Tutorial dan panduan mengenai element <code>&lt;legend&gt;...&lt;/legend&gt;</code> yang digunakan untuk memberi judul &lt;fieldset&gt; element. Mencakup penjelasan yang disertai contoh kode penggunaan sebagai rujukan untuk materi belajar HTML <span lang="id">elemen</span> &lt;legend&gt;.
</p>
<hr class="uk-article-divider">

<h2 class="title-sub bd-danger bd-left bd-left-only">Deskripsi <br>
    <small>Penjelasan HTML <span class="highlight">legend</span></small>
</h2>
<p>
  <strong>HTML <code>&lt;legend&gt;</code> element</strong> merepresentasikan sebuah judul bagi konten induknya, <code>&lt;fieldset&gt;</code> element secara keseluruhan. Beberapa control dalam sebuah form dapat dikelompokkan didalam <code>fieldset</code> element dan diberi judul dengan <code>&lt;legend&gt;</code>.
</p>
<p>Dalam penggunaannya, elemen <code>&lt;legend&gt;</code> ditulis secara langsung (tanpa didahului element lain) sebagai anak element dari &lt;fieldset&gt;.</p>

<hr class="uk-article-divider">

          <p>Silahkan, lihat referensi mengenai <a href="/tutorial/html/html-global-attribute.html">HTML Global Attribute</a></p>

    <small>Atribut event  (Peristiwa)</small>
  </h2>
    <div class="dul-callout dul-callout-warning">
        <p>&lt;legend&gt; tag mencakup <em>event attributes</em>, yang artinya attribute tersebut dijalankan ketika ada interaksi dari user atau dalam suatu peristiwa (kejadian). Contoh: menjalankan script (JavaScript) ketika halaman web pada jendela browser hendak ditutup, dan lain sebagainya.</p>
