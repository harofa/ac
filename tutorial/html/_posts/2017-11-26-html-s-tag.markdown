<p class="text-muted">
    Referensi Belajar <strong>HTML s tag</strong>. Tutorial dan panduan mengenai element <code>&lt;s&gt;...&lt;/s&gt;</code> yang digunakan untuk menyatakan teks yang sudah tidak benar atau tepat. Mencakup pembahasan lebih detail dan lengkap yang disertai contoh kode penggunaan sebagai rujukan untuk materi belajar HTML <span lang="id">elemen</span> &lt;s&gt;.
</p>
<hr class="uk-article-divider">

<h2 class="title-sub bd-danger bd-left bd-left-only">Deskripsi <br>
    <small>Penjelasan HTML <span class="html-tag">s</span> tag</small>
</h2>
<p>HTML <code>s</code> tag menunjukkan sebuah <em>strikethrough</em> atau <em>garis coret</em>.</p>
<p>
  <strong>HTML <code>&lt;s&gt;</code> element</strong> digunakan untuk menunjukkan konten atau teks yang sudah tidak lagi akurat atau relevan (berkaitan).
</p>
<p><code>&lt;s&gt;</code> element akan menghasilkan <em>style</em> yang sama percis dengan element <code>&lt;del&gt;</code>, yaitu teks tersebut terdapat garis tengah lurus yang menunjukkan <em>tercoret</em>, adapun penggunaan kedua element tersebut jelas berbeda.</p>
<p><code>&lt;s&gt;</code> element, tidak tepat digunakan ketika menunjukkan perubahan (edit) dokumen yang telah dihapus, untuk hal ini, gunakan <a href="https://www.apacara.com/tutorial/html/html-del-tag.html" title="HTML del tag">&lt;del&gt;</a> sebagai gantinya (karena lebih tepat dalam penggunaannya).</p>

  <h2 class="title-sub bd-danger bd-left bd-left-only">Event Attributes <br>
    <small>Atribut event  (Peristiwa)</small>
  </h2>
    <div class="dul-callout dul-callout-warning">
        <p>&lt;s&gt; tag mencakup <em>event attributes</em>, yang artinya attribute tersebut dijalankan ketika ada interaksi dari user atau dalam suatu peristiwa (kejadian). Contoh: menjalankan script (JavaScript) ketika halaman web pada jendela browser hendak ditutup, dan lain sebagainya.</p>
