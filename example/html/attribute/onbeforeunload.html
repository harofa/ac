demo HTML onbeforeunload attribute

Mengenai onbeforeunload

onbeforeunload adalah HTML Event Attribute yang digunakan untuk menjalankan script sebelum halaman ditutup (unload). Biasanya digunakan untuk memberi peringatan seperti: "Apakah yakin halaman mau ditutup?",dan sebagainya.

Petunjuk:

Cobalah untuk menutup (close), atau reload halaman ini pada browser Anda, dialog box akan muncul meminta perhatian.
