Logo apacara.com disamping ini aslinya terletak pada relative URL: "/media/images/apacara.com.png" akan tetapi, disini hanya ditulis nama filenya saja (apacara.com.png) atau lebih jelasnya img src="apacara.com.png". Kenapa bisa demikian? karena kita sudah menuliskan elemen base dengan rujukan link dasarnya yaitu: "/media/images/" (value dari href attribute pada base element tersebut), sehingga kita tidak perlu menulis secara lengkap cukup nama file yang merujuk pada base url tersebut .

Kunjungi apacara.com. Link ini akan terbuka pada jendela baru, karena pada element base kita sudah menentukan "target=_blank" yang artinya dibuka pada jendela/tab baru untuk semua URL dasarnya.
